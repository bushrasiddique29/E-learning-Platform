# 📚 E-Learning Platform (MERN Stack)

A full-stack **E-Learning Platform** built using the MERN stack (MongoDB, Express, React, Node.js).  
It allows users to browse courses, enroll, watch lectures, and complete payments securely.

---

## 🚀 Features

### 👨‍🎓 User Side
- User authentication (Signup/Login with JWT)
- Browse available courses
- View course details & lectures
- Enroll in courses
- Secure payment integration (Razorpay/Stripe)
- Track enrolled courses

### 🧑‍💻 Admin Side
- Add / update / delete courses
- Upload lectures/videos
- Manage users & enrollments

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Context API / Redux (if used)
- Axios
- React Router
- Tailwind CSS / CSS

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Razorpay Payment Gateway
